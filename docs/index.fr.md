# Menu geOrchestra

Le header geOrchestra est un composant web (`geor-header`) base sur Vue qui
fournit un en-tete et une barre de navigation communs aux applications.


![image info](./images/menu_overview.png)

## Ce que fait le composant

- Branding et navigation coherents entre applications
- Menus, login/logout et libelles i18n configurables
- En-tete legacy en iframe si besoin

## Usage rapide

```html
<script src="https://cdn.jsdelivr.net/gh/georchestra/header@dist/header.js"></script>
<geor-header></geor-header>
```
